# PENSUM DE COMPETENCIAS: CREACIÓN DE SVGS CREATIVOS

## PERFIL DE EGRESO
Al finalizar este curso, el estudiante será capaz de:
1. Diseñar gráficos vectoriales escalables directamente desde código para optimizar rendimiento y control.
2. Implementar sistemas de iconos y visualizaciones responsivas utilizando `viewBox` y coordenadas relativas.
3. Crear experiencias de usuario enriquecidas mediante animaciones performantes en SVG y CSS.

## MATRIZ DE COMPETENCIAS POR MÓDULO

| Módulo | Competencia Específica (Saber Hacer) | Resultado de Aprendizaje (Evidencia) |
| :--- | :--- | :--- |
| **Módulo 0** | Configurar y dimensionar lienzos vectoriales. | SVG responsive que se adapta a cualquier contenedor sin deformarse. |
| **Módulo 1** | Dibujar geometrías arbitrarias con precisión matemática. | Réplica exacta de un logotipo geométrico usando solo `<path>`. |
| **Módulo 2** | Aplicar composición avanzada y teoría del color vectorial. | Ilustración con texturas, sombras (gradientes) y recortes complejos. |
| **Módulo 3** | Animar propiedades vectoriales para narrativa visual. | Componente de UI (ej. botón o loader) con micro-interacción fluida. |
