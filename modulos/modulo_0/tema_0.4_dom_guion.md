### 00:00 - INTRODUCCIÓN

Llegamos al concepto final: El DOM.
Si vienes del diseño gráfico, esto te volará la cabeza.

---

### 00:30 - ESTÁ VIVO

**[LOCUTOR]**:
En SVG, cada círculo, cada línea, cada letra sigue viva dentro del navegador.
Son "nodos del DOM", igual que un párrafo de texto o un botón.
¿Qué significa esto?
Que puedes usar CSS para cambiarles el color cuando el usuario pasa el mouse.
Que puedes usar JavaScript para hacer que reboten cuando les haces clic.

---

### 01:00 - CAJA DE CRISTAL

**[LOCUTOR]**:
Un JPG es una caja fuerte cerrada. No puedes tocar lo de adentro.
Un SVG es una caja de cristal abierta. Puedes meter la mano, reorganizar los muebles y volver a salir.
Esa interactividad es el verdadero poder del SVG en la web moderna.
